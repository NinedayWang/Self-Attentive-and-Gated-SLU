# 基于 Pytorch 复现模型 **Self Attentive and Gated SLU**

由于需求比较急，完成时间紧张，代码质量比较粗糙。

## 运行环境

> + Python 2.7
> + Pytorch 0.3*

## Demo 模型

在主目录节点下，命令 

      python main.py
      
即可，代码会自动适配 GPU/CPU。

## 参数控制

源码文件 main.py 中以 Java 常数命名规则命名的变量均是可调节的超参数。
